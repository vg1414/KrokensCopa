# Krokens Copa

Ett tipptävlingssystem för en sluten grupp. Deltagarna tippar matchresultat och samlas i en gemensam topplista.

**Live:** https://vg1414.github.io/KrokensCopa/

## Funktioner

- **Matcher** – Adminvy för att lägga upp kommande matcher
- **Tippa** – Välj spelare och lämna tippning på varje match
- **Låsning** – Tippningar låses när matchen startar
- **Topplista** – Poängräkning med leaderboard-tabell
- **Omgångar** – OS och PL (separata turneringar)

## Teknik

- Vanilla HTML/CSS/JavaScript – inga ramverk
- Mobilanpassad med flikar (Matcher / Tippa / Topplista)
- Lila/blå gradient-design
